# AbelDB
This is my NoSQL database created using Python.

## Dependencies
* Python3.14

## Commands
For Lint Check, run the following:
```commandline
ruff check
```

For Lint Format, run the following:
```commandline
ruff format .
```

For order code in file, run the following:
```commandline
sdsort <your_file.py>
```
